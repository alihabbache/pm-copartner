The 5 Agents
01

Discovery

Turns raw user interview notes into a structured insight report — themes, quotes, and ranked opportunities.

Run when: After research sprints, before planning
02

Planning

Scores and ranks your backlog against your OKRs. Every item gets a score and a one-line rationale.

Run when: Weekly before sprint planning
03

Specs

Auto-drafts a PRD from any feature request — even a single sentence. Flags every gap as an open question.

Run when: On demand, any new request
04

Analytics

Compares your metrics to targets, assigns RAG status, surfaces top movers, and recommends actions.

Run when: Weekly Monday or Friday EOD
05

Launch

Writes user-facing release notes from your sprint output — in your brand voice, for your audience.

Run when: Per sprint close / per release
