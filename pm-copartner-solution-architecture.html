Step-by-Step: How to Run Any Agent
1
Choose your agent
Open the folder for the workflow you need — e.g. 01-discovery/ after a round of user interviews, or 04-analytics/ on Monday morning.
2
Read setup.md once
Decide your path: Minimal (paste data into Claude — zero setup) or Connected (MCP integrations for auto pull/push). You only do this once per agent.
3
Open prompt.md and copy the full prompt
Each prompt is self-contained and ready to use. Paste it into Claude Desktop or claude.ai.
prompt.md → select all → copy
4
Fill the [bracket] placeholders
Replace [PASTE YOUR NOTES HERE] with your actual data. Delete the input options you're not using. The fewer brackets left, the better the output.
5
Run Claude
Send the message. Claude processes the input and returns a structured document — insight report, ranked backlog, PRD, metrics digest, or release notes.
6
Save or route the output
Copy to Notion, Google Docs, email, Slack, or save as a file. If you included a save instruction in the prompt (MCP path), this happens automatically.
7
(Optional) Automate with /schedule
In Claude Desktop, type /schedule after your prompt to set a recurring run. Recommended cadence: Planning + Analytics weekly, Specs always-on, Launch per release.
